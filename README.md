# Codveda Internship Projects

This repository contains the projects completed during the **Codveda Software Development Internship Program**. The projects demonstrate practical skills in full-stack web development, database integration, authentication, API development, GraphQL, React, Node.js, Express, MongoDB, and modern software engineering practices.

## Internship Overview

The Codveda Internship Program provided hands-on experience in designing, developing, and deploying modern web applications. Each project focused on a specific area of software development, progressing from foundational concepts to advanced full-stack applications.
